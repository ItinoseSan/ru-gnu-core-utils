# ru-gnu-core-utils
# Usage
### 1.clone this repository and move to the directory.
```
git clone git@github.com:ItinoseSan/ru-gnu-core-utils.git
```
```
cd ru-gnu-core-utils
```
### 2.Set up use crate
```
cargo build
```
### 3.Run src
```
cargo run
```
